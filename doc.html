Minor Notes on Lurnn ALPHA:
  • Successfully registered/notice of confirmation email sent needs to look more professional (not just an alert box on a blank page, use template)
  • Change “Quiz Name” to “Learning Goal” and add “Quiz Name"
  • After quiz is taken and they, redirect to class dashboard (unless it is self-assessment vs. assessment, if it is, add button to take them to Class Dashboard)
  • On progress bar, doesn't actually say what grade is... show in box what actual percent is
  • TYPOS: "Rest password" "Assined classes”, should be “Reset Password” “Assigned Classes"
  • "Class code" should be optional, not required to create class
  • won't let you change profile picture once enrolled
  • won't let student go to profile once there are no quizzes left to take
  • "meeting hours" not a meaningful item, unless used to order the classes on dashboard by which classes teacher teaches
  • Timestamp/date students progress so it can be tracked
  • Teachers should be allowed to change number of possible answers on quiz


Lurnn BETA:
  • Teachers would like to duplicate class quizzes if teaching same subject
  • Give students the opportunity to retake quizzes and/or take new quizzes of the same topic to measure progress towards learning goal. (Teacher must give permission for student to re-take quiz)
  • Give teachers ability to provide feedback to students once they have taken quiz, comments on each question and/or overall quiz
  • Having math type or equation editor to write fractions, numbers under radicals, etc. for math class quizzes
  • Add ability for teachers to create True/False questions and questions where the answer is an input text box and/or a paragraph answer (essay)
  • Show student progress per skill/ allow to re-assess on same skills and track improvement using a line graph (not a bar graph)
  • Teachers find bar graph useful for showing class by skill, so they know which skills they still need to focus on. Line graphs are needed to show progress over time.
  • Teachers should be able to create learning goals for classes
  • For example:
  • Learning goal: Students will be able to use ratios and proportions to solve problems.
  • Then she can create a scale (usually 4 points, but it would be nice to have it flexible for each learning goal)
  • 0: I cannot write a ratio
  • 1: I can write a ratio for a given situation
  • 2: I can write a ratio and determine if two ratios are proportional
  • 3: I can write ratios, determine if they are proportional, and solve to find values that will make two ratios proportional
  • 4: I can write ratios, solve proportions, and write equations for proportional linear relationships
  • Then the student would be able to login and select where he/she feels that he falls on the scale at that time, and it would be recorded with a date/timestamp. Students can rate themselves as often as necessary and view a tracking chart/ line graph (kind of like the stock market) and they can see how it changes over time…
  • The teacher’s end needs to be able to view students’ self-evaluation scores and chart maybe as a bar graph by class and the # of students at each level.
  • It could be multiple choice, and then if the student selects the correct answer, it could recommend that the student is at that respective level.
  • Also cool if teachers could share rating scales between each other, so they don’t have to recreate everything if they’re teaching the same course.
  • Also teachers need to be able to enter grades such as classwork/quiz/test grades and those could be tracked as well…

It would be cool if the teacher could add questions to the scale that the student could try to answer to help them self-evaluate.
Example:
1: Ms. Eike has 10 blue marbles, 3 red marbles, and 17 white marbles. What is the ratio of red marbles to white marbles?
2: Jeremy earns $317 for 36 hours of work. Miguel earns $342 for 40 hours of work. Are the pay rates for both people proportional?
3: If 5 cans of juice will be sufficient for 8 people on a weekend campout, how many cans will be needed for 56 people?
4: If 4 cups of sugar makes 18 cookies, write a linear equation for the relationship between cups of sugar and cookies.
